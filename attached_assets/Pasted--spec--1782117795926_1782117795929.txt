Κάτω είναι το δέντρο σε καθαρό κείμενο. Είναι spec — διόρθωσέ/κόψε/πρόσθεσε ό,τι θες.

Σημειογραφία πεδίων:
```
:text     μονή γραμμή            :date    ημερομηνία
:rtext    πολυγραμμικό/richtext   :bool    ναι/όχι
:num      αριθμός                 →X       αναφορά σε άλλο node X
:select{a|b|c}  μονή επιλογή       [X]      λίστα από child-nodes X (repeater)
:multi{a|b|c}   πολλαπλή επιλογή   :calc    υπολογιζόμενο
?         προαιρετικό            ⊢cond     κανόνας/condition
```

```
ΒUILDER ΝΟΜΙΚΩΝ ΚΕΙΜΕΝΩΝ
│
├─ 0. SHARED NODES (value-nodes, επαναχρησιμοποιήσιμα παντού)
│   ├─ Φορέας
│   │   ├─ Όνομα            :text
│   │   ├─ Είδος            :select{Υπουργείο|Ανεξάρτητη Αρχή|ΝΠΔΔ|ΟΤΑ|Λοιπό}
│   │   └─ Προϊστάμενος?    :text
│   ├─ Πρόσωπο (υπογράφων)
│   │   ├─ Ονοματεπώνυμο    :text
│   │   ├─ Ιδιότητα         :text
│   │   └─ Φορέας           →Φορέας
│   └─ Παραπομπή (citation/reference)
│       ├─ Στόχος           →(Νόμος|ΠΔ|ΥΑ|Άρθρο|ΦΕΚ|ΕΕ-Πράξη)
│       ├─ Είδος σχέσης     :select{αναφορά|τροποποιεί|καταργεί|αντικαθιστά|
│       │                            κατ’εξουσιοδότηση|κωδικοποιεί|κυρώνει}
│       └─ Εμβέλεια?        :text   (π.χ. «άρθρο 5 παρ.2»)
│
├─ 1. ΔΟΜΙΚΑ NODES (building blocks — συνθέτουν τα κείμενα, αναδρομικά)
│   ├─ Μέρος
│   │   ├─ Αριθμός          :text   (Α΄, Β΄…)
│   │   ├─ Τίτλος           :text
│   │   └─ Περιεχόμενο      [Κεφάλαιο | Άρθρο]
│   ├─ Κεφάλαιο/Τμήμα
│   │   ├─ Αριθμός          :text
│   │   ├─ Τίτλος           :text
│   │   └─ Περιεχόμενο      [Άρθρο]
│   ├─ Άρθρο
│   │   ├─ Αριθμός          :text   (5, 5Α…)
│   │   ├─ Τίτλος?          :text
│   │   ├─ Κατάσταση        :select{Ισχύον|Καταργηθέν|Αντικατασταθέν|Προστεθέν}
│   │   │      ⊢cond Κατάσταση∈{Καταργηθέν|Αντικατασταθέν} ⇒ εμφάνισε «Με βάση»
│   │   ├─ Με βάση?         →Παραπομπή
│   │   └─ Περιεχόμενο      [Παράγραφος]
│   ├─ Παράγραφος
│   │   ├─ Αριθμός          :text   (1, 2…)
│   │   ├─ Κείμενο          :rtext
│   │   └─ Περιεχόμενο?     [Περίπτωση]
│   ├─ Περίπτωση
│   │   ├─ Στοιχείο         :text   (α, β, γ…)
│   │   ├─ Κείμενο          :rtext
│   │   └─ Περιεχόμενο?     [Υποπερίπτωση]
│   └─ Υποπερίπτωση
│       ├─ Στοιχείο         :text   (αα, ββ…)
│       └─ Κείμενο          :rtext
│
├─ 2. ΦΕΚ (container δημοσίευσης)
│   ├─ Τεύχος               :select{Α΄|Β΄|Γ΄|Δ΄|ΑΑΠ|ΥΟΔΔ|ΠΡΑΔΙΤ|ΔΔΣ|ΑΣΕΠ}
│   ├─ Αριθμός φύλλου       :num
│   ├─ Έτος                 :num
│   ├─ Ημερομηνία           :date
│   └─ Δημοσιεύει           [→Νόμος|→ΠΔ|→ΥΑ|→ΠΝΠ]
│
└─ 3. TEMPLATES ΚΕΙΜΕΝΩΝ
    │
    ├─ Νόμος
    │   ├─ Αριθμός          :text   (4624/2019)
    │   ├─ Τίτλος           :text
    │   ├─ Είδος            :select{Τυπικός|Κυρωτικός|Κωδικοποιητικός|Ερμηνευτικός}
    │   │      ⊢cond Είδος=Κυρωτικός ⇒ εμφάνισε «Κυρούμενο κείμενο» →ΕΕ-Πράξη/Σύμβαση
    │   ├─ Ημ. ψήφισης      :date
    │   ├─ ΦΕΚ              →ΦΕΚ
    │   ├─ Έναρξη ισχύος    :select{Από δημοσίευση|Από ημερομηνία|Όπως ορίζει το ίδιο}
    │   │      ⊢cond =Από ημερομηνία ⇒ εμφάνισε «Ημ. έναρξης» :date
    │   ├─ Κατάσταση        :select{Σε ισχύ|Τροποποιημένος|Καταργημένος|Κωδικοποιημένος}
    │   ├─ Συσχετίσεις      [Παραπομπή]   ← το «interaction»: τροποποιεί/καταργεί άλλα
    │   └─ Περιεχόμενο      [Μέρος | Κεφάλαιο | Άρθρο]
    │
    ├─ ΠΝΠ (Πράξη Νομοθετικού Περιεχομένου)
    │   ├─ Τίτλος           :text
    │   ├─ Ημ. έκδοσης      :date
    │   ├─ ΦΕΚ              →ΦΕΚ
    │   ├─ Κύρωση με νόμο?  →Νόμος
    │   └─ Περιεχόμενο      [Άρθρο]
    │
    ├─ Προεδρικό Διάταγμα (ΠΔ)
    │   ├─ Αριθμός          :text
    │   ├─ Τίτλος           :text
    │   ├─ Νομική βάση      [Παραπομπή]   (εξουσιοδοτικός νόμος)
    │   ├─ Εισηγητής φορέας →Φορέας
    │   ├─ ΦΕΚ              →ΦΕΚ
    │   └─ Περιεχόμενο      [Άρθρο]
    │
    ├─ Απόφαση (ΥΑ / ΚΥΑ)
    │   ├─ Τύπος            :select{ΥΑ|ΚΥΑ|Κανονιστική}
    │   │      ⊢cond Τύπος=ΚΥΑ ⇒ «Εκδίδοντες» γίνεται multi (≥2 Φορείς)
    │   ├─ Αριθμός          :text
    │   ├─ Θέμα             :text
    │   ├─ Εκδίδοντες       →Φορέας  (ΥΑ) | [→Φορέας] (ΚΥΑ)
    │   ├─ Νομική βάση      [Παραπομπή]  ← trigger: «κατ’εξουσιοδότηση άρθρου Χ»
    │   │      ⊢offer αν η βάση δείχνει υπαρκτό →Άρθρο ⇒ πρότεινε σύνδεση/άντληση
    │   ├─ Υπογράφοντες     [→Πρόσωπο]
    │   ├─ ΦΕΚ              →ΦΕΚ
    │   └─ Περιεχόμενο      [Άρθρο]
    │
    ├─ Εγκύκλιος
    │   ├─ Αρ. πρωτοκόλλου  :text
    │   ├─ ΑΔΑ?             :text
    │   ├─ Θέμα             :text
    │   ├─ Εκδίδων          →Φορέας
    │   ├─ Σχετικά          [Παραπομπή]
    │   ├─ Ημερομηνία       :date
    │   └─ Κείμενο          :rtext
    │
    └─ Κωδικοποίηση (codified node — δεν είναι «νέο» κείμενο, είναι όψη)
        ├─ Βασικό κείμενο   →(Νόμος|ΠΔ)
        ├─ Ενσωματώνει      [Παραπομπή]   (όλες οι τροποποιήσεις)
        ├─ Είδος            :select{Επίσημη|Ανεπίσημη/de facto}
        ├─ Ενημ. έως        :date
        └─ Παραγόμενο κείμ. :calc   (auto από βασικό + τροποποιήσεις)
```

Τα `⊢cond` είναι τα ενδο-node conditions και τα `⊢offer` τα δια-node interactions. Πες μου τι λείπει ή τι αλλάζει.
